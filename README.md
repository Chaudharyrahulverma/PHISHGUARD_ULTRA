# PHISHGUARD ULTRA

AI-powered phishing detection and cybersecurity analysis platform designed to detect malicious URLs, phishing emails, suspicious content, and risky passwords using machine learning and AI techniques.

---

## 🚀 Features
- 🔍 URL Phishing Detection
- 📧 Email Threat Analysis
- 🖼 OCR-based Image Scanning
- 🤖 AI Cybersecurity Assistant
- ⚠️ Content Risk Detection
- 🔐 Password Strength Analyzer
- 📊 Security Reporting System

---

## 🛠 Tech Stack
- Python
- Flask
- JavaScript
- HTML/CSS
- SQLite
- OCR
- Machine Learning
